# Darkade / Murderland Arcade

A static browser arcade game built with HTML, CSS, and JavaScript. Clear waves, fight bosses, collect powerups, and chase a local high score.

## Run Locally

Open `index.html` in a browser, or serve the repository root with any static file server.

## Deploy

This project is ready for Netlify as a static site:

- Publish directory: repository root
- Build command: none

## Files

- `index.html` - Game markup and SEO metadata
- `style.css` - Responsive arcade UI styling
- `game.js` - Canvas game logic and controls
